# rsj_projects
Walmart Confidence interval

Dataset link: https://d2beiqkhq929f0.cloudfront.net/public_assets/assets/000/001/293/original/walmart_data.csv?1641285094

About Project:

• Analyzed 550K+ Walmart Black Friday transactions to understand customer purchasing
   behavior across demographics such as gender, age, and marital status.
   
• Performed exploratory data analysis (EDA) using Python libraries like Pandas, Seaborn,
   and Matplotlib to identify trends in spending and product demand.      
   
•  Applied statistical techniques including hypothesis testing (t-test, ANOVA) and chi-square    
    tests to validate differences in purchasing behavior.
    
•  Built confidence interval simulations using Central Limit Theorem to compare average 
    transaction values across customer segments.
    
•  Identified key business insights such as high-demand product categories and higher 
    average spending across Gender, Marital Status and Age.
    
• Provided data-driven recommendations to optimize inventory, marketing strategies, and 
    city-level store expansion.
